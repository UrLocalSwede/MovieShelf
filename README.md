# Movie Browser

MovieVault is a desktop application that scans local and network drives for movie files, organizes them into a searchable library, and provides one-click playback through VLC Media Player. Designed for users with large personal movie collections who want a fast and simple viewing experience.


Run the app with:

1. Double-click `dist/MovieBrowser.exe`, or
2. Double-click `run_app.bat`, or
3. Run `C:/Users/Swede/.local/bin/python3.14.exe app.py`

The app scans `Documents\Movies` for `.mkv` and `.mp4` files and opens them with VLC if available, or your system default player otherwise.
