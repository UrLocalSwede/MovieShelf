# Movie Browser

Run the app with:

1. Double-click `dist/MovieBrowser.exe`, or
2. Double-click `run_app.bat`, or
3. Run `C:/Users/Swede/.local/bin/python3.14.exe app.py`

The app scans `Documents\Movies` for `.mkv` and `.mp4` files and opens them with VLC if available, or your system default player otherwise.
